to scroll when zoomed in we need to set start_sample end_sample accordingly

break down of this:
signalY1 = ((DISP_H * buffer[(signalX + 1) * ((end_sample - start_sample) / (DISP_W * x_zoom_factor / 100)) + 1]) / 4096) * (y_zoom_factor / 100) + y_pos;
- DISP_H * BUFFER / 4096 -> normalize and expand it to whole vertical space
- * y_zoom_factor / 100 -> vertical zoom (remove the paranthesis to stop getting it fucked, its int type)
- + y_pos offset
- ((end_sample - start_sample) / (DISP_W * x_zoom_factor / 100)) -> bin the window to whole horizontal range,
	x_zoom_factor will expand the range so we get tighter bins
- (signalX + 1) * choose Nth sample from the bin
so choose a window, bin it, choose a sample from bin, normalize and strech, add offset. (I'd do the same thing tbh)

x_zoom_factor is like Tdiv coefficient, y_zoom_factor is Vdiv coefficient.
the 100 thing is to give finer control over factors.
